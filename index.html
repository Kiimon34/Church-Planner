💡 Formatting Key Guide

Use these structural keys to format entries perfectly across the application tabs:

🗓️ Monthly Calendar View Inputs
Entries: Separate multiple unique events inside a single box using commas ,.
Example: 10:00 AM Worship, Youth Study, Evening prayer
✨ Blessings Tab Inputs
Format: Name : Description : Status Separate the member's name, their thanksgiving item, and their badge status flag using colons :.
Available Status Flags:
  • Full (Renders a green success badge)
  • Low (Renders a yellow caution badge)
  • Empty (Renders a red alert badge)
Example: John Doe : Healing Recovery : Full
