💡 Formatting Key Guide

When modifying events directly on calendar cells, use these distinct syntax symbols to organize rows automatically across the dashboards:

📢 Announcements: Start any line with a 📢 emoji.
Example: 📢 Church picnic this Sunday!
⏱️ Activities: Separate time and event with a single dash -.
Example: 9:00 AM - Morning Devotion
📋 Preaching: Separate Team, Time, and Location with vertical bars |.
Example: Team Alpha | 11:00 AM | Sanctuary
🛠️ Duties / Roles: Separate the duty assignment and name with a colon :.
Example: Usher: Brother John

Admin Controls Panel

Format: Name : Description : Status (Full, Low, Empty)
